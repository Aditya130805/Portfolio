Spotify "Advert Muter"

Mutes all Spotify advertisements!

How did I do it?
The process was simple:
  1. Get the currently playing track
    • The Spotify API was used to get the link to the data containing the currently playing track.
    • A request was sent to that URL - the authorization code (a.k.a. the access token) was obtained beforehand from Spotify for Developers.
    • The JSON response was stored in a variable.
    • From the dictionary formed in the variable, the currently playing track was obtained.
  2. Identify whether it is a song or an advert
    • If the current track was a song, a valid result was returned.
    • If the current track was an advertisement, the program resulted in an error.
    • Hence, a try, except block was utilized to identify whether the track playing was a song or an advertisement.
  3. Reduce/Increase the device volume based on whether it was a song or an advert
    • The device volume was modified via 'applescript'. Note that this implies that the program only runs on MacOS.
    • Set volumes were decided for both songs and advertisements.
    • Volumes were accordingly changed via the 'applecript' module in python.
